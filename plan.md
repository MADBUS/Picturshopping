# 전자제품 이미지 인식 & 쇼핑 검색 서비스

## 프로젝트 개요
- Gemini Vision API로 전자제품 사진 분석
- 네이버 쇼핑 API로 유사 상품 검색
- CLI에서 이미지 경로 입력 → 분석 결과 + 쇼핑 검색 결과 출력

## 기술 스택
- 언어: Python
- LLM: Gemini 2.5 Flash (Vision 지원)
- 검색: 네이버 쇼핑 API
